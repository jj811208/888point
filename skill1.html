Gulp.js 簡易入門

前端自動化工具

講者: 初振言

前端工程過去所遇到的問題

  1. 瀏覽器百家爭鳴 -> postCSS
  2. JS新標準的兼容性差 -> babel
  3. CSS樣式重覆與難以維護 -> SASS , CSS-Module
  4. 網站超肥 套件多 -> minifyCSS , uglify , imagemin
    5. ......

為什麼需要Gulp.js?

懶為了工作流的最佳化!!

原本前端工程師的工作流程

  1. 確認需求
  2. 寫程式(html,css,js)
  3. 使用pug編譯html
  4. 使用sass編譯css
  5. 使用postcss再編譯一次css
  6. 使用babel編譯js
  7. 使用minifyCSS壓縮css
  8. ......
  9. debug

使用Gulp.js後的工作流程

  1. 確認需求
  2. 寫程式(html,css,js)
  3. 編寫gulpfile.js
  4. 執行task
  5. debug

如何使用-安裝篇

打開cmd 
//確認是否有安裝npm
npm -v

//第一次安裝
npm i -g gulp

//確認是否安裝成功
gulp -v

//將gulp安裝在你的專案
cd /path/to/yourproject/
npm i gulp
然後在這目錄中新建一個gulpfile.js

如何使用-指令篇

//cmd
gulp 'taskname' //執行任務

//gulpfile.js
task('xxx',function) //定義一個任務

src() //來源檔案

pipe() //流程

dest() //輸出檔案的位置

如何使用-情境篇_壓縮html

先移動到專案資料夾 安裝相關套件 
npm install --save-dev gulp-htmlmin
打開gulpfile.js 
//gulpfile.js
var gulp = require('gulp');
var htmlmin = require('gulp-htmlmin');

gulp.task('minify', function() {
  return gulp.src('html/*.html')
    .pipe(htmlmin({collapseWhitespace: true}))
    .pipe(gulp.dest('dist/html'));
});
回到cmd 
gulp minify

如何使用-情境篇_串起工作流

打開gulpfile.js 
gulp.task('abc', function() {
	--some code--
});
gulp.task('hello', function() {
	--some code--
});
gulp.task('Money', function() {
	--some code--
});

gulp.task('gogogo'',["abc","hello","Money"]);
回到cmd 
gulp gogogo
